# 版本修改记录

说明：仅记录 1.3.0 及以上版本；最新版本置于前面。以下内容按视图归类，并标记为 Add / Fix。未标版本号的提交，已并入对应的较新版本。

## 1.3.11
- Peek 视图
  - Fix：当定义结果包含 `targetRange` 时，优先定位到定义起始位置，避免 `typedef` / `enum` 等声明类符号停在末尾。

- Symbol视图
  - Add：也添加了限定文件范围的功能。

## 1.3.10
- Map 视图
  - Add：支持右键菜单，可复制符号名、文件路径等信息。
  - Add：Outline 列表支持类名单行 / 多行 / 隐藏三种显示方式。
  - Add：Outline 支持调整列宽。
  - Add：Outline 选中行高亮显示。
  - Fix：优化地图视图显示，区分背景与编辑器区域。

## 1.3.9
- Peek 视图
  - Add：符号有多定义场景下，提供列表进行选择性预览。

## 1.3.8
- Map 视图
  - Add：支持通过分析按钮右侧的 File 选项，限定或排除分析结果中的文件。

## 1.3.7
- Map 视图
  - Add：支持水平 / 垂直分区显示。
  - Add：移除底部操作提示，界面更简洁。

## 1.3.6
- Map 视图
  - Add：增加标签右键菜单。
  - Add：支持多实例，每个实例可独立配置视图选项。
  - Fix：修复 Ctrl + 左键无法展开 / 收起节点的问题。

## 1.3.5
- Peek 视图
  - Add：增加环形历史缓存，支持更接近浏览器前进 / 后退的历史体验。
  - Add：增加锁定按钮，可临时屏蔽编辑器光标更新。

## 1.3.4
- 视图通用
  - Add：更新三个视图的图标。
  - Add：增加效果预览。

## 1.3.3
- Symbol Search 视图
  - Add：支持多关键字搜索。
  - Add：补全所有符号类型对应的 emoji 图标。

## 1.3.2
- 公共
  - Add：抽取三个视图的公共代码，便于后续维护。

## 1.3.1
- Symbol Search 视图
  - Add：结果支持语法高亮。
  - Add：支持单击行为配置。
- Map 视图
  - Add：支持单击行为配置。

## 1.3.0
- Symbol Search 视图
  - Add：新增符号查询视图。
  - Add：支持精确 / 模糊搜索模式切换。
  - Add：支持结果排序方式切换。
- Map 视图
  - Add：优化节点布局，父节点会随展开的子节点自动调整位置。
